# Laser Pointer, 1998

Интерактивный 3D-сайт: китайская лазерная указка-брелок из 90-х — жёлтый глянцевый
корпус, золотые насадки с узорами, наклейка «DANGER», три батарейки LR41 и цепочка.
Всё управляется мышью, вся геометрия и текстуры процедурные (canvas + lathe),
звуки синтезируются WebAudio на лету. Никаких внешних ассетов.

## Запуск

```bash
npm install
npm run dev
```

## Как играть

- **Тяните указку** — она вращается как физический предмет. Тяните пустое место — вращается камера.
- **Зажмите латунную кнопку** — из насадки бьёт красный луч, на бетонной стене загорается узор.
  Пока кнопка зажата, мышь водит лучом. При первом включении узор сначала складывается в «HELLO».
- **Кликните переднюю насадку** и крутите мышью **против часовой** — она честно выкрутится по
  резьбе и уляжется в коробку. Кликните насадку в коробке — подлетит; закрутите по часовой
  (или кликните ещё раз — закрутится сама). Четыре узора: сердечки, звёзды, сетка, спираль.
- **Кликните задний колпачок** и открутите его тем же жестом. Наклоните указку задним концом
  вниз — батарейки выскользнут. Без батареек кнопка просто щёлкает. Кликните батарейку — она
  вернётся в отсек; закрутите колпачок — лазер снова жив.

## Стек

Vite + React + TypeScript, three.js через @react-three/fiber + drei
(Lightformer-окружение вместо скачиваемых HDR), Bloom из @react-three/postprocessing,
переходы на @react-spring/three, физика цепочки — verlet-верёвка, батареек — простая
ручная интеграция. Состояние — явная машина в `src/stateMachine.ts`.

Структура: `Pointer.tsx`, `Nozzle.tsx`, `NozzleBox.tsx`, `BatteryBay.tsx`, `Chain.tsx`,
`Beam.tsx`, `Wall.tsx`, `GestureManager.tsx`, `stateMachine.ts`, `patterns.ts`, `textures/`.
