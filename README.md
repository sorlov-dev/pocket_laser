# Laser Pointer

Интерактивный 3D-сайт: китайская лазерная указка-брелок из 90-х — жёлтый глянцевый
корпус, золотые насадки с узорами, наклейка «DANGER», три батарейки LR41 и цепочка.
Указка парит на чёрном фоне, всё управляется мышью. Вся геометрия и текстуры
процедурные (canvas + lathe), звуки синтезируются WebAudio на лету. Никаких внешних ассетов.

**Живой сайт:** https://sorlov-dev.github.io/pocket_laser/

## Запуск

```bash
npm install
npm run dev
```

## Как играть

- **Тяните указку** — она вращается как физический предмет. Тяните пустое место — вращается камера.
- **Зажмите латунную кнопку** — камера уезжает «из-за плеча», и из насадки бьёт красный луч,
  а в пустоте перед дулом загорается узор текущей насадки. При первом включении узор сначала
  складывается в «HELLO». Отпустили — камера возвращается.
- **Насадки** — список слева (сердечки, звёзды, сетка, спираль): клик меняет узор мгновенно.
  Или кликните переднюю насадку и крутите мышью **против часовой** — она честно выкрутится
  по резьбе и слетит.
- **Кликните задний колпачок** и открутите его тем же жестом. Наклоните указку задним концом
  вниз — батарейки выскользнут. Без батареек кнопка просто щёлкает. Кликните батарейку — она
  вернётся в отсек; закрутите колпачок — лазер снова жив.

## Стек

Vite + React + TypeScript, three.js через @react-three/fiber + drei
(Lightformer-окружение вместо скачиваемых HDR), Bloom из @react-three/postprocessing,
переходы на @react-spring/three, физика цепочки — verlet-верёвка, батареек — простая
ручная интеграция. Состояние — явная машина в `src/stateMachine.ts`.

Структура: `Pointer.tsx`, `Nozzle.tsx`, `BatteryBay.tsx`, `Chain.tsx`, `Beam.tsx`,
`Projection.tsx`, `GestureManager.tsx`, `UI.tsx`, `stateMachine.ts`, `patterns.ts`, `textures/`.

## Деплой

Пуш в ветку `main` автоматически собирает и публикует сайт через GitHub Actions
(`.github/workflows/deploy.yml`, GitHub Pages). `base` в `vite.config.ts` завязан на
имя репозитория (`/pocket_laser/`) — при переименовании репо не забудьте поправить.
